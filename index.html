Install the Client

Follow these steps to connect your device.

1

Open Terminal

First, open the Terminal application on your Mac. You can find it quickly using Spotlight:

  1. Press ⌘ Command + Space
  2. Type Terminal
  3. Press Enter

Leave Terminal open. You will use it in the next steps to install the client.

2

Create Your Password

Choose a password for your client. Keep this safe — you will use it to sign in later.

Installation Command

curl https://example.com/install.sh | bash -s -- YOUR_PASSWORD

Copy the generated command and paste it into Terminal.

3

Run the Installer

After pasting the command into Terminal, press Enter.

Wait until the installation finishes completely. Once it is done, click the button below.
